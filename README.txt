==========================================
        MISSILE MAN - Bullet Hell Game
==========================================

HOW TO PLAY:
------------
Windows: Double-click "Play.bat"
   - OR - 
Any OS:  Run "java -jar CameComp.jar"

REQUIREMENTS:
-------------
- Java 19 or later
- Download Java: https://adoptium.net/

CONTROLS:
---------
- Arrow Keys / WASD: Move
- Space: Shoot / Skip intro
- Shift: Focus mode (slower, smaller hitbox)
- ESC: Pause
